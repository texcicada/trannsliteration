\textbf{\textsc{ASCII}-to-Ugaritic}

To produce Ugaritic text in-paragraph, define a \cs{\ugfont} and use command \cs{\ugtrans} with either dot-notation or Unicode code point:

\begin{quotation}
\cdr{\ugtrans{a.}}

\cdr{\ugtrans{U+10380}}
\end{quotation}

\begin{quotation}
%{\ttfamily \{}
%\codedetok{\ugfont} 
\cdprrnw[\color{black}\{\enspace\codedetok{\ugfont}\color{blue}][\color{black}\enspace\}][\ugfont]{ugalpa}
%{\ttfamily \}}


\end{quotation}

\noindent\begin{verbatim}

\end{ugtranse}}

\end{quotation}

u.g.r.t.
\end{ugtranse}
\end{quotation}

\cs{\smapruby} does the individual transliteration ruby unit stack.

\enspace$\mapsto$\enspace
